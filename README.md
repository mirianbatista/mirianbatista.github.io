# mirianbatista.github.io

Site pessoal em HTML/CSS estático, servido pelo GitHub Pages a partir da branch `main`. O blog vive na pasta `blog/` deste repositório.

## Estrutura

- `index.html` — sobre, experiência, projetos
- `css/style.css` — paleta derivada de Classic Blue `#0F4C81`, light/dark via `prefers-color-scheme`
- `images/` — fotos
- `404.html` — página de erro própria

Ao alterar conteúdo, atualizar a data no rodapé das páginas.
